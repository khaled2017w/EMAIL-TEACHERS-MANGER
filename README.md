# Gmail Manager (Email Teachers Manager)

A desktop app to send and delete emails using Gmail.  
Built with Python + Tkinter.

## Requirements
- Gmail account with 2‑Factor Authentication enabled
- An **App Password** (generate at myaccount.google.com)

## How to run
1. Install Python 3.7+
2. Run `python gmail_manager.py`
3. Enter your Gmail address and App Password

## Security
- No credentials are stored in the code.
- You must enter your App Password each session.
