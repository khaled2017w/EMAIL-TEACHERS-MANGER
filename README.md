A desktop app to send and delete emails using Gmail.  
Built with Python + Tkinter.
<img width="1487" height="1160" alt="image" src="https://github.com/user-attachments/assets/d357cb11-d4e8-4833-ba66-20e2c555419c" />

## Requirements
- Gmail account with 2‑Factor Authentication enabled
- An **App Password** (generate at myaccount.google.com)
<img width="1481" height="1154" alt="image" src="https://github.com/user-attachments/assets/d093e30c-ce7d-47ff-a46d-d693cf18c418" />

## How to run
1. Install Python 3.7+
2. Run `python gmail_manager.py`
3. Enter your Gmail address and App Password
<img width="866" height="751" alt="image" src="https://github.com/user-attachments/assets/9c5241f3-8548-4b90-a9e0-eb5d049ea8ce" />

## Security
- No credentials are stored in the code.
- You must enter your App Password each session.
